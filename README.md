# 📉 Telecom Customer Churn Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing customer attrition (churn) within a telecommunications company. Using a dataset of **7,043 customers**, I built an interactive Excel dashboard to identify key drivers of churn and provide data-driven recommendations to improve retention strategies.

## 🚀 Key Business Insights
After deep-diving into the data, I identified three critical factors influencing customer churn:

| Insight Category | Findings | Recommendation |
| :--- | :--- | :--- |
| **Tech Support** | Customers without Tech Support have a **42% churn rate**. | Bundle Tech Support with high-tier internet plans. |
| **Contract Type** | **Month-to-month contracts** are the primary churn drivers. | Incentivize long-term (1-2 year) contracts with loyalty rewards. |
| **Payment Method** | Manual payment (Electronic Check) shows higher churn than Auto-pay. | Encourage Auto-pay adoption through small monthly discounts. |
| **Service Tier** | **Fiber Optic** users churn more despite premium speeds. | Investigate service stability and price-to-value perception. |

## 🛠️ Skills & Tools Used
* **Data Cleaning:** Handling missing values and formatting raw data.
* **Excel Pivot Tables & Charts:** Aggregating data for visual storytelling.
* **Key Performance Indicators (KPIs):** Calculating Churn Rate, Retention Rate, and Monthly Revenue Loss.
* **Dashboard Design:** Implementing Slicers, Custom Formatting, and Interactive Markers.

---

## 📈 Conclusion
The goal of this analysis is to transform raw telecom data into **Actionable Business Intelligence**. By focusing on the first 12 months of the customer journey (Tenure) and securing manual payment users into automated systems, the company can significantly reduce revenue loss.
